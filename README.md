# Mutual Fund Performance Analysis: Outlier Detection and Diversification

A comprehensive study investigating the identification and behavioral patterns of outlier mutual fund managers across multiple sectors over a 27-year period (1999-2025).

## Overview

This repository contains the code and data for analyzing sector-specific mutual fund performance using daily NAV returns. The analysis employs residual-based classification, quantile tracking, and Hamming distance metrics to identify outlier funds and assess portfolio diversification across Energy, Healthcare, Real Estate, Technology, and Utilities sectors.

## Key Features

- **Market Model Regression**: Isolates fund-specific performance from broader market movements by regressing against S&P 500
- **Quantile Classification**: Tracks fund migration between performance deciles over time
- **Hamming Distance Analysis**: Quantifies behavioral distinctiveness and diversification contributions
- **Crisis Period Analysis**: Examines fund behavior during dot-com crash (2001), financial crisis (2008), and COVID-19 pandemic (2020-2021)
- **Sector-Level Comparisons**: Analyzes both intra-sector and inter-sector diversification patterns

## Methodology

### Analysis Pipeline

```
┌─────────────┐
│   main.py   │  ← Step 1: Download fund data
└──────┬──────┘
       │ Outputs: funds/*, logs/*
       ↓
┌─────────────┐
│residuals.py │  ← Step 2: Compute market-adjusted residuals
└──────┬──────┘
       │ Outputs: residuals/*, logs/*
       ↓
┌─────────────┐
│ main.ipynb  │  ← Step 3: Quantile classification & analysis
└──────┬──────┘
       │ Outputs: quantiles/*, plots/*
       ↓
   [Results]
```

### 1. Market Model Regression
For each fund *i* at observation *k*:

```
R(i,k) = α_i + β_i * R_m(k) + ε(i,k)
```

Where:
- `R(i,k)` = fund return
- `R_m(k)` = market benchmark return (S&P 500)
- `α_i` = fund-specific excess return (alpha)
- `β_i` = systematic risk
- `ε(i,k)` = idiosyncratic residual

### 2. Decile Classification
Residuals are sorted and divided into 10 quantiles (1-10):
- **Quantile 1**: Bottom 10% (closest to market behavior)
- **Quantile 10**: Top 10% (farthest deviation from market)

### 3. Hamming Distance
Normalized Hamming distance measures how often two funds occupy different quantiles:

```
d_H(Q_i, Q_j) = (1/T) * Σ I(q_i,t ≠ q_j,t)
```

Where `I(·)` is an indicator function and `T` is the number of time periods.

## Usage

### Complete Analysis Pipeline

The analysis follows a three-step workflow:

#### Step 1: Data Collection
```bash
python main.py
```
- Downloads fund data from Yahoo Finance
- Saves data to `funds/` directory organized by sector
- Creates processing logs in `logs/` directory

#### Step 2: Residual Computation
```bash
python residuals.py
```
- Loads fund and benchmark data
- Performs market model regression (fund returns vs S&P 500)
- Computes and saves residuals to `residuals/` directory
- Generates regression statistics and logs

#### Step 3: Quantile Analysis & Visualization
```bash
jupyter notebook main.ipynb
```
- Open the notebook and run all cells
- Classifies residuals into deciles (quantiles)
- Saves quantile classifications to `quantiles/` directory
- Generates all visualizations and saves to `plots/`:
  - Quantile trajectory plots
  - Hamming distance heatmaps
  - Crisis period analysis
  - Diversification contribution tables

### Analyzing Individual Funds

For single-fund diagnostic analysis:
```bash
python single_fund.py
```

## Key Findings

### Energy Sector
- Overall Hamming distance: **0.658**
- Best diversifier: **PRNEX** (Drop: 0.051)
- High synchronization during oil price shocks
- Strong commodity exposure dominates performance

### Healthcare Sector
- Overall Hamming distance: **0.734** (highest)
- Best diversifiers: **ETHSX, FBDIX** (Drop: 0.013 each)
- Greater internal diversification than other sectors
- Subsector heterogeneity drives differentiation

### Real Estate Sector
- Overall Hamming distance: **0.563** (lowest)
- Best diversifier: **DPREX** (Drop: 0.018)
- Strong interest rate sensitivity
- High correlation during credit crises
